Modifiers

Groups
[abc] - any char between
[^abc] - any char NOT between
[0-9] - any digit
[x|y] - any alternative listed
(?...) - passive (non-capturing)
\x - Group/subpattern x
Meta
. - single char (- newline or line terminator)
\w - word
\W - non-word
\d \D - digits
\s \S - whitespace
\b \B - beginning
\0 - NUL
\n \f \r - new line \ form feed \ carriage return
\t \v - tab \ vertical tab
\xxx - char by octal num xxx
\xdd - hex
\uxxxx - unicode
Quantifier
n+ - at least one n
n* - zero or more n
n? - zero or one n
n{X} - sequence of X n's
n{X,Y} - sequence X to Y n's
n{X,} - sequence at least X n's
n$ - n at end
^n - n at start
?=n - followed by n
?!n - not followed by n
String Replacement
$n - nth non-passive
$` - Before matched
$' - After matched
$+ - last matched
$& - Entire matched
